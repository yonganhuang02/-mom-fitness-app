# 妈妈减肥打卡

一款轻量级的每日体重打卡应用，支持体重记录、打卡鼓励语、目标体重设置。

## 功能

- 📅 **每日打卡**：记录当天体重并完成打卡
- 💬 **随机鼓励语**：每次打卡后显示暖心鼓励语
- 📊 **数据统计**：累计打卡天数、当前体重、目标进度
- 🎯 **目标设置**：设置目标体重，查看减肥进度
- 📋 **历史记录**：查看过往体重记录

## 使用方法

### 本地使用

1. 直接双击 `index.html` 用浏览器打开
2. 或在项目目录下运行简易服务器（推荐，避免部分浏览器本地文件限制）：
   ```bash
   # Python 3
   python -m http.server 8080
   # 然后访问 http://localhost:8080
   ```

### 手机使用

1. 将项目部署到服务器或使用 GitHub Pages
2. 用手机浏览器打开链接
3. 在浏览器菜单中选择「添加到主屏幕」即可像 App 一样使用

## 数据存储

数据保存在浏览器 LocalStorage 中，无需网络，隐私安全。
